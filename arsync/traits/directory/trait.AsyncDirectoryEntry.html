Expand description

Unified directory entry interface for both local and remote operations

This trait provides a consistent interface for directory entries regardless of whether the underlying storage is local (using compio-fs-extended) or remote (using protocol transports like SSH, rsync, etc.).

§Examples

ⓘ
use arsync::traits::AsyncDirectoryEntry;

let entry = directory.read_entries().await?.into_iter().next().unwrap();
println!("Entry name: {}", entry.name());
let metadata = entry.metadata().await?;
println!("Is file: {}", metadata.is_file());
