pub async fn copy_file_with_trait(src: &Path, dst: &Path) -> Result<u64>
Expand description

Copy a file using the AsyncFile trait

This is a simple demonstration of using the trait-based file abstraction for file copying. It uses the AsyncFile trait methods to read from source and write to destination.

§Parameters

  • src - Source file path
  • dst - Destination file path

§Returns

Returns Ok(bytes_copied) on success, where bytes_copied is the total number of bytes copied from source to destination.

§Errors

Returns Err(SyncError) if:

  • Source file cannot be opened or read
  • Destination file cannot be created or written
  • I/O error occurs during copy

§Examples

ⓘ
use arsync::copy_trait::copy_file_with_trait;
use std::path::Path;

#[compio::main]
async fn main() -> arsync::Result<()> {
    let bytes = copy_file_with_trait(
        Path::new("source.txt"),
        Path::new("dest.txt")
    ).await?;
    println!("Copied {} bytes", bytes);
    Ok(())
}
